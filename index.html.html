Guide de mise en place

Étapes pour activer l'envoi d'emails

  • 1
    Créez un compte gratuit sur emailjs.com (200 emails/mois offerts)
  • 2
    Ajoutez un Email Service (Gmail, Outlook…) dans "Email Services"
  • 3
    Créez 2 Email Templates : un pour vous notifier, un pour le dentiste. Utilisez les variables : {{nom}}, {{email}}, {{telephone}}, {{materiel}}, {{type_demande}}, {{description}}, {{urgence}}
  • 4
    Récupérez votre Public Key dans Account → API Keys
  • 5
    Hébergez ce fichier HTML sur GitHub Pages (gratuit) ou tout autre hébergeur, puis copiez l'URL dans l'onglet QR Code
🔧 Variables disponibles pour vos templates EmailJS :

{{nom}} · {{email}} · {{telephone}} · {{materiel}} · {{type_demande}} · {{urgence}} · {{description}} · {{email_depot}}

📱 Notification smartphone : Activez les notifications email sur votre téléphone, ou ajoutez une règle de filtre "Priorité haute" sur les emails venant du formulaire.
